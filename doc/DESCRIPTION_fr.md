# Fonctionnalités de SABnzbd

SABnzbd est un lecteur de news binaire gratuit et simple pour télécharger, vérifier et extraire des fichiers depuis Usenet.

## Fonctionnalités principales

- **Traitement automatique** : Télécharge, vérifie, répare et extrait les fichiers automatiquement
- **Interface web** : Interface web moderne et responsive pour une gestion facile
- **Multi-plateforme** : Fonctionne sur Linux, Windows, macOS et de nombreuses autres plateformes
- **Support RSS** : Téléchargement automatique basé sur les flux RSS
- **Catégories** : Organisez les téléchargements avec des catégories personnalisables
- **Post-traitement** : Décompression automatique et organisation des fichiers
- **Contrôle de bande passante** : Limitez les téléchargements pour préserver les ressources réseau
- **Sécurisé** : Support HTTPS/SSL pour des connexions sécurisées
- **Extensible** : Système de plugins pour des fonctionnalités supplémentaires

## Formats supportés

- Archives RAR (avec réparation automatique via PAR2)
- Archives ZIP
- Archives 7-Zip
- Archives TAR/GZ

## Intégration

Ce paquet YunoHost inclut :

- Gestion automatique du service
- Rotation des journaux
- Intégration du répertoire multimédia
- par2cmdline-turbo optimisé pour des réparations plus rapides
