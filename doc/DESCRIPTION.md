# SABnzbd Features

SABnzbd is a free and easy binary newsreader for downloading, verifying and extracting files from Usenet.

### Key Features

- **Automatic Processing**: Downloads, verifies, repairs and extracts files automatically
- **Web Interface**: Modern, responsive web interface for easy management
- **Multi-platform**: Works on Linux, Windows, macOS and many other platforms
- **RSS Support**: Automatic downloading based on RSS feeds
- **Categories**: Organize downloads with customizable categories
- **Post-processing**: Automatic unpacking and file organization
- **Bandwidth Control**: Throttle downloads to preserve network resources
- **Secure**: HTTPS/SSL support for secure connections
- **Extensible**: Plugin system for additional functionality

### Supported Formats

- RAR archives (with automatic repair using PAR2)
- ZIP archives
- 7-Zip archives
- TAR/GZ archives

### Integration in YunoHost

- Automatic service management
- Log rotation
- Multimedia directory integration
- Optimized par2cmdline-turbo for faster repairs
